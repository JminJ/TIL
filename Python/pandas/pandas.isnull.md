# pandas.isnull()
---

## 형태 및 설명

- pandas.isnull(obj)
- array-like object의 value가 비어있는 값을 추적한다.
- 이 함수는 scalar 또는 array-like object를 가지고 value가 NaN인 것을 가리킨다.

## Parameters

- obj(scalar or array-like) - null 또는 missing value를 체크할 Object

## Returns

- bool or array-like of bool
