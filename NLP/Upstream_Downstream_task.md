# Upstream/Downstream Task
---

## upstream task

upstream task는 BERT나 GPT처럼 대규모 학습 모델에서 **pre-train을 수행한 모델**을 의미한다.

## downstream task

downstream task는 대규모 학습 모델에서 **fine-tuning을 수행한 모델**을 의미한다.
