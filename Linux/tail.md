# tail
-------
> 파일의 내용 중 뒷부분을 기준으로 보여준다.

## 문법 & 대표 옵션

- tail option file
- -n : 뒤에서부터 몇 개의 line을 출력한다.
- -c : 뒤에서부터  몇 bytes의 내용을 출력한다.
- -f : 내용이 새롭게 추가될 경우 그 내용을 보여준다.

## 사용법

```bash
tail 파일명

tail -n 파일명
```
