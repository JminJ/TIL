# head
------
> 파일의 내용 중 앞 부분을 몇 줄 보여준다.

## 문법 & 옵션

- head option file
- -n : line을 기준으로 보여준다.
- -c : bytes들을 기준으로 보여준다.

## 사용법

```bash
head 파일명

head -n 100 파일명
```
